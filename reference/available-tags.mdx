## Tag Naming Convention

Tags follow the pattern: `<php-version>-<type>[-<variant>][-<extension>]`

- **php-version**: PHP major.minor version (8.1, 8.2, 8.3, 8.4)
- **type**: Either `cli` or `fpm`
- **variant**: Optional `alpine` for Alpine Linux-based images (Debian Bookworm otherwise)
- **extension**: Optional `swoole` for images with Swoole extension

## Image Updates

All images are automatically rebuilt and retagged weekly based on their corresponding upstream PHP images to ensure security updates and bug fixes are included.
