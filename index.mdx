## Quick Start

Get started with a simple pull and run:

<CodeGroup>

```bash Docker Pull
docker pull namoshek/php-mssql:8.4-cli-alpine
```

```bash Docker Run
docker run -it namoshek/php-mssql:8.4-cli-alpine php -v
```

```bash Docker Compose
services:
  app:
    image: namoshek/php-mssql:8.4-fpm-alpine
    volumes:
      - ./app:/var/www
```

</CodeGroup>
